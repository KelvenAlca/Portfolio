MEU TRABALHO

Ciência de Dados: Uma Breve Introdução

Sou o autor do livro "Ciência de Dados: Uma Breve Introdução", uma obra abrangente que visa introduzir os leitores ao mundo da ciência de dados e programação. Do básico ao avançado, conduzo os leitores por uma jornada educativa que abrange matemática, estatística, linguagens de programação, análise de dados, aprendizado de máquina e muito mais. Com uma linguagem clara e acessível, meu livro foi desenvolvido para ajudar os leitores a desenvolverem habilidades essenciais e se destacarem no campo da ciência de dados, seja você um iniciante em busca de conhecimento ou um profissional procurando aprimorar suas habilidades.

Londrina Dados 2023

Sou o autor do projeto "Londrina Dados 2023", uma análise financeira detalhada da Prefeitura de Londrina para o ano de 2023. Meu objetivo foi identificar padrões e áreas de melhoria na gestão financeira, utilizando dados oficiais sobre receitas, despesas, investimentos e parcerias. Alguns resultados destacados incluem a diferença entre receita prevista e arrecadação final, análise das despesas fixas, investimentos realizados pela prefeitura e parcerias estratégicas.

ML: Previsão de Nota Animes

Esse projeto de Machine Learning foi meu primeiro após estudar a fundo o tema, onde utilizei o banco de dados "Anime Recommendations Database" do Kaggle e fiz uma análise descritiva das características dos animes com base nas informações do MyAnimeList. Aprofundando-me, desenvolvi um modelo para prever a nota de um anime, considerando diversas características fornecidas pelos usuários, marcando assim minha entrada no mundo da análise preditiva e da inteligência artificial aplicada ao entretenimento.

Eleições Brasileiras

Projeto de pesquisa que analisa a relação entre as eleições brasileiras e a meritocracia, utilizando a teoria de 5% de diferença meritocrática. Os dados eleitorais foram coletados manualmente na internet desde a Nova República, permitindo a análise dos resultados para diferentes cargos. A metodologia inclui a aplicação da teoria meritocrática para compreender como a distribuição de mérito entre os candidatos influencia os resultados eleitorais. Os resultados esperados incluem insights sobre padrões e tendências nas eleições brasileiras, contribuindo para o debate sobre a aplicação da meritocracia na política do país.

Criptografia e Descriptografia em Python

Script em Jupyter Notebook demonstra de forma simples o processo de criptografia e descriptografia em Python, utilizando a biblioteca cryptography. O objetivo é apresentar como esses conceitos podem ser implementados de maneira eficiente e segura, oferecendo uma visão prática do funcionamento da criptografia em Python.

MNIST : Reconhecimento de Dígitos

Um projeto de reconhecimento de dígitos usando Machine Learning, utilizando a base de dados MNIST e a biblioteca TensorFlow. O modelo treinado alcançou alta precisão na classificação de dígitos de 0 a 9, proporcionando uma valiosa experiência em Machine Learning.
